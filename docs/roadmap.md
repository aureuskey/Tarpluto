# Tarpluto Roadmap

## Project Vision
Merge biological computing with human cognition to create a non‑invasive Neural Patch + Living Node wet‑ware processor that enables thought‑speed intelligence, energy‑efficient human‑AI symbiosis, and a new paradigm of augmenting minds without surgery.

## Current Status
Solo founder with laptop + Wi‑Fi. Core simulation v0.4 implements typing‑based Neural Patch, Living Node wet‑ware processing, persistent memory, symbiotic AI suggestions, command system, and auto‑saved visualizations. Ready to iterate toward real‑world wet‑ware and BCI integration.

## 30‑Day Insane Deadline Plan (Week by Week)

**Week 1 – Foundations & Signal Fidelity**
- Day 1‑2: Replace mock NeuralPatch with real‑time keyboard inter‑key‑interval analysis (compute dwell time, flight time, burst detection).
- Day 3‑4: Calibrate signal mapping to physiological ranges (0‑1 µV‑like) and add noise model.
- Day 5‑7: Validate that sustained Enter presses produce ramp‑up signal; log and plot signal vs. think‑time.

**Week 2 – Living Node Realism**
- Day 8‑9: Increase neural‑state resolution to 400 neurons; implement short‑term synaptic depression/facilitation.
- Day 10‑11: Add refractory period and spike‑threshold detection; output becomes spike count per bin.
- Day 12‑14: Visualize raster plot alongside heat‑map; ensure stability under varied input regimes.

**Week 3 – Symbiotic AI Intelligence**
- Day 15‑16: Integrate a tiny local LLM (e.g., TinyLlama or Phi‑2 via llama.cpp) for generative suggestions; fallback to rule‑based.
- Day 17‑18: Prompt engineering: ask model for “one creative insight + one concrete next step” based on user input + node output.
- Day 19‑21: Latency <2 s on laptop; cache responses; add simple safety filter.

**Week 4 – Wet‑ware Bridge & Demo**
- Day 22‑23: Design minimal hardware interface spec (e.g., OpenBCI GPIO → simulated wet‑ware stimulus).
- Day 24‑25: Write a mock “FinalSpark API” wrapper that accepts neural spike trains and returns metabolic cost.
- Day 26‑28: End‑to‑end loop: think → NeuralPatch → Living Node → API → motor output (simple cursor or LED blink).
- Day 29‑30: Record a 60‑second demo video, polish README, and push release v0.5.

## Key Milestones
- **M1 (Day 7):** Real‑typing NeuralPatch with calibrated signal.
- **M2 (Day 14):** Living Node exhibiting spike‑based output and short‑term plasticity.
- **M3 (Day 21):** Symbiotic AI powered by on‑device LLM delivering actionable insights.
- **M4 (Day 30):** Closed‑loop bio‑digital demo simulating wet‑ware interaction; public demo video.
